# Meu portfólio

Site que fiz pra juntar meus projetos e formas de contato num lugar só. É só
HTML, CSS e JavaScript, sem framework nenhum.

**Ver online:** https://luisguilherme605.github.io/portfolio-luis-guilherme/

## O que tem

- uma apresentação e um pouco da minha trajetória
- seção de projetos com link pra cada site no ar
- contato (email, GitHub, LinkedIn, WhatsApp)
- funciona bem no celular e no desktop
- uns detalhes que curti fazer: cursor com rastro, fundo de estrelas no canvas,
  animações de fade ao rolar e menu hambúrguer no mobile

## Rodando localmente

```bash
git clone https://github.com/LuisGuilherme605/portfolio-luis-guilherme.git
cd portfolio-luis-guilherme
```

Depois é só abrir o `index.html` no navegador (ou usar o Live Server no VS Code).
